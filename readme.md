# Use it
make an env.json
```
{
  "NAME": "USERNAME",
  "PORT": 8000,
  "IP": "127.0.0.1"
}
```
have python running the server by opening main.py and selecting host. Or have someone run it and have their port forwarded IP in there. Have a client open main.py and join as client!
